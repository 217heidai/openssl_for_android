# OpenSSL for Android
Automatically compile static OpenSSL(3.0.*) library for Android by Github Actions.

## Android
`*MIPS` targets are no longer supported with NDK R17+.
* [x] armeabi
* [x] armeabi-v7a
* [x] arm64-v8a
* [x] x86
* [x] x86_64
* [ ] mips
* [ ] mips64
