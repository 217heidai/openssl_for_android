# OpenSSL for Android
Automatically compile static OpenSSL library for Android by Github Actions.

Added `-fPIC` to fix linking error.

## Android
`armeabi`、`mips`、`mips64` targets are no longer supported with NDK R17+.
* [ ] armeabi
* [x] armeabi-v7a
* [x] arm64-v8a
* [x] x86
* [x] x86_64
* [ ] mips
* [ ] mips64
