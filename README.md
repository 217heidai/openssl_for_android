# OpenSSL for Android
Automatically compile static OpenSSL(1.1.1*) library for Android by Github Actions.

## Android
* [x] armeabi
* [x] armeabi-v7a
* [x] arm64-v8a
* [x] x86
* [x] x86_64
* [x] mips
* [x] mips64
